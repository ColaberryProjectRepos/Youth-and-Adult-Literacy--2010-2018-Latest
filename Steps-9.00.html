Transforming the Data in Power Query Editor 9
  • In this step you will clean and transform the data
  •    Select Transform Data from Home tab.
  • Transform the Year column:
    • Duplicate Year column and rename the duplicate column ‘End of Year’. 
    • Set the datatype for this column as Date. Then select the column and transform it into ‘Year’ and then ‘End of Year’. 
    • In the Home ribbon, go to Transform and change the Data Type to Text and then to Date. Select Add New Step. 
    • Right click on the End of Year column, select Transform, Year, and then End of Year. The data type will be set to Date. 
  • Select the Literacy rate column and change the datatype to Fixed Decimal number. In the Gender column remove the total row from the drop down button. 
  • Rename your table to ‘Literacy’.
  • Click on file and select Close and Apply. Follow the gif below step by step.



Add Continent dataset and Gender-Short column
  1. On Power BI Desktop ribbon, Click on Transform data
  2. Select Merge Queries and select the ‘Contrycontinent’ table. The Literacy table is already selected.
  3. Choose the Country column from both tables and click ok.
  4. Expand the newly added table and select only Country and Continent columns. 
  5. Rename the columns as CountryName and ContinentName as shown in the GIF.
  6. Duplicate Gender column and rename the duplicate column Gender-Short. Replace the value ‘female’ by ‘F’ and ‘male’ by ‘M’
