Creating Measures Table 12

1.From Power BI desktop ribbon, Home tab click on Enter Data.
2.Name the table ‘ Measures/All_Measures’ and the column ‘Measures’ as well. 
3.Start creating measures after right clicking on the column measure and then selecting ‘New measure’. 
4. Fill in all the DAX formulas provided below, one at a time, and when you finish Press Enter or select Commit (checkmark icon) in the formula bar. 
5. After creating each measure choose the necessary format and decimal for each value (measure) then save. 
Follow the gif and create the table and create the measures as columns in the All_Measures table. 
Measures (Dax Formula)

1. Total Literacy = 
SUM('LITERACY'[Literacy rate])
2. AvgLiteracyRate = 
AVERAGE(LITERACY[Literacy rate])
3. Literacy% = 
DIVIDE(SUM(LITERACY[Literacy rate]), COUNTROWS(LITERACY))
4. Literacy YoY% =
 VAR TotalLiteracy_Rate_CY= Average(Literacy[Literacy Rate]) 
VAR Lit_LYr = CALCULATE ( AVERAGE( Literacy[Literacy Rate] ),
SAMEPERIODLASTYEAR ( 'Calendar'[Date]) )
Return
IF(ISBLANK(Lit_LYr),0,Divide(TTLit_CY-Lit_LYr,TTLit_CY))
5. Lit_Indicator YoY% = SWITCH(TRUE(), [Literacy_YoY%]<0,
UNICHAR(9662), [Literacy_YoY%] =0, UNICHAR(9666), UNICHAR(9652))
6. Total MALE Literacy =
 CALCULATE([Total Literacy],FILTER(LITERACY,[Gender] = "Male") )
7. Male Literacy Rate% = 
IF(ISBLANK(AVERAGEX(FILTER('Literacy', 'Literacy'[Gender] = "male"),
'Literacy'[Literacy rate])), 0, AVERAGEX(FILTER('Literacy', 'Literacy'[Gender] = "male"),
'Literacy'[Literacy rate]))
8. M Lit YoY% =
Var MLitPY= CALCULATE(AVERAGEX(FILTER('Literacy', 'Literacy'[Gender] = "male"),
'Literacy'[Literacy rate]), 'Literacy'[Year] = MAX('Literacy'[Year]) - 1)
Var MLitCY=CALCULATE(AVERAGEX(FILTER('Literacy', 'Literacy'[Gender] = "male"),
'Literacy'[Literacy rate]), 'Literacy'[Year] = MAX('Literacy'[Year]))
RETURN IF(MLitPY=0, 0, (MLitCY-MLitPY)/MLitPY)
9. M_Indicator YoY% = 
SWITCH(TRUE(), [M Literacy YoY_%] <0, UNICHAR(9662),[M Literacy YoY_%]=0,
UNICHAR(9666), UNICHAR(9652))
10. Total FEMALE Literacy = 
CALCULATE([Total Literacy],FILTER(LITERACY,[Gender] = "Female") )
11. Female Literacy Rate% = 
IF(ISBLANK(AVERAGEX(FILTER('Literacy', 'Literacy'[Gender] = "female"),
'Literacy'[Literacy rate])), 0, AVERAGEX(FILTER('Literacy', 'Literacy'[Gender] = "female"), 'Literacy'[Literacy rate]))
12. F_Indicator YoY% = 
SWITCH(TRUE(), [Female Literacy Rate Year over Year Percentage] <0,
UNICHAR(9662),[Female Literacy Rate Year over Year Percentage] =0, UNICHAR(9666), UNICHAR(9652))
13. F Lit YoY% =
Var FLitPY= CALCULATE(AVERAGEX(FILTER('Literacy', 'Literacy'[Gender] = "female"),
'Literacy'[Literacy rate]), 'Literacy'[Year] = MAX('Literacy'[Year]) - 1)
Var FLitCY=
CALCULATE(AVERAGEX(FILTER('Literacy', 'Literacy'[Gender] = "female"),
'Literacy'[Literacy rate]), 'Literacy'[Year] = MAX('Literacy'[Year]))
RETURN IF(FLitPY=0, 0, (FLitCY-FLitPY)/FLitPY)
14. M Lit Adv = 
CALCULATE([Male Literacy Rate]-[Female_Literacy_Rate])
